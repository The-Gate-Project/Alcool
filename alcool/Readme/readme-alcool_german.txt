Alkohol Mod für BG2
by Mathrim Cauthon

Dieser Mod fügt Gespräche zwischen den Charakteren am Ende von Kapitel 3 hinzu, wenn der Spieler mit Aran Linvail vereinbart zur Zauberfeste aufzubrechen. (kurz bevor die Szene mit dem Ausbruch von Irenicus gezeigt wird)
